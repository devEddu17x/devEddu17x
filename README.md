## What I do

- **Scalable Backends & APIs**: Build robust services and communication channels with Node.js, NestJS, WebSockets, and WebRTC (P2P).
- **Infrastructure as Code (IaC)**: Design and provision AWS and Cloudflare environments using modular, environment-safe Terraform configurations.
- **Production DevOps**: Automate continuous delivery (CI/CD) pipelines via GitHub Actions and OIDC role-assumed authentication, operating with Docker containers and Linux systems.
- **Data & Distributed Workflows**: Implement reliable storage solutions (Amazon S3, Cloudflare R2) and distributed data pipes (Apache Kafka, PySpark).
